# SimpleChatNorp
Flask-based web chat with a C system agent for real-time desktop notifications


## System dependencies (Linux)
sudo apt install libcurl4-openssl-dev libcjson-dev libnotify-bin

## System dependencies (Windows)
pacman -S mingw-w64-x86_64-curl mingw-w64-x86_64-cjson

## Python requirements
Flask==3.0.3
Werkzeug==3.0.3
